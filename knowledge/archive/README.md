# Pre-canonical knowledge archive

This archive preserves the `v0.1.0` and `v0.2.0` development packs and their original type-oriented layout. Archived content is immutable history, not an active knowledge authority, and is never runtime eligible.

The original manifests are retained verbatim. Their asset paths record the former repository layout; `archive_index.yml` and the canonical `v0.3.0` provenance references identify the current archive location.

New knowledge must not be added here. The only canonical content authority is `knowledge/packs/<pack_id>/<version>/manifest.yml`.
