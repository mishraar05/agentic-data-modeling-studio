# Public US P&C Personal Auto knowledge pack v0.4.0

This immutable candidate succeeds `v0.3.0` without rewriting it. It corrects the California Personal Auto jurisdiction overlay by using the applicable Insurance Code chapter, distinguishing financial responsibility from insurance coverage, preserving material claims-timing exceptions, and preferring official California sources.

The correction does not increase the claimed content completeness. The manifest is the sole pack authority, modules cannot be selected by directory scanning, and all content remains candidate and runtime-ineligible until named owners and required reviewers approve an exact immutable version through the governed registry workflow.
